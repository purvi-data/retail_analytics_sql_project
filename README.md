# retail_analytics_sql_project
Retail Sales Analytics project using SQL for business insights and customer segmentation.


## Objective
Analyze sales data to understand product performance, customer behavior, and revenue trends.

## Key Insights
- Top 10% customers contribute significantly to total revenue.
- Clothing category drives highest revenue growth.
- Revenue shows volatility with one major 116% MoM spike.
- High-value products generate strong revenue despite lower unit sales.

## Tools Used
- MySQL
- Data Cleaning using SQL
- Window Functions
- CTEs
- Aggregations

## Business Impact
Insights can help improve marketing focus, inventory planning, and customer targeting.
